# 快捷键说明
ctrl + N :根据类名查找。
ctrl + shift + N :根据文件名查找。
ctrl + shift + F :查找文件内容，范围是整个项目中的所有文件，可以过滤文件类型、大小写、完整词组等。
ctrl + F :查找文件内容，范围是当前文件，可以过滤大小写、完整词组、选择的代码等。
ctrl + shift + R :替换文件内容，范围是所有文件，可以过滤文件类型、大小写、完整词组等。
ctrl + R ::替换文件内容，范围是当前文件，可以过滤大小写、完整词组、选择的代码块等。
ctrl + alt + L :格式化代码，默认格式化长点的代码会强制换行，需要设置代码行长度，见:File -> Settings -> Editor -> Code Style -> Hard wrap at : 800。
ctrl + alt + O :清理import。
ctrl + Q :查看方法说明。前提是光标放在方法上才行，javascript同样适用。
ctrl + - :折叠当前代码块
ctrl + + :展开当前代码块
ctrl + shift + - :折叠当前文件所有代码块
ctrl + shift + + :展开当前文件所有代码块
alt + enter :在红色波浪线处使用，一般用于导入包，接口实现等。
alt + insert :自动生成插入getter，setter，接口实现等。前面这几个一般在光标位置处生成，前提是光标位置是正确的（比如光标不能在方法内部）。
alt + 光标 :整列选中，选中后可以整列删除、粘贴、复制等
